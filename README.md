# Demographic-Engagement-Analysis
Python program that filters user data by age and country, calculates platform and demographic statistics, and analyzes correlation between age and online influence.

The script generates four outputs:
> HIGHLY ENGAGED USERS :
    Users from the selected country
    Age within the specified range
    Verified accounts
    More than 7 hours spent
    Returns user ID and influence score
> LIST OF UNIQUE COUNTRIES FOUND IN THE DATASET.
> DEMOGRAPHIC STATISTICS: 
    Mean income
    Standard deviation
    Which region (urban/rural/suburban) has the lowest engagement time
> CORRELATION BETWEEN AGE AND INFLUENCE ON INSTAGRAM :
    Calculated using Pearson correlation formula
